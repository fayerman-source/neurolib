# Code of Conduct

This project adopts the **Contributor Covenant, version 2.1**, as its code of conduct.

The full text of the covenant is available at:
https://www.contributor-covenant.org/version/2/1/code_of_conduct/

By participating in this project — opening issues, submitting pull requests, or engaging in discussion — you agree to abide by its terms.

## Scope

This Code of Conduct applies within all project spaces (this repository, its issue tracker, pull requests, and any associated discussion channels) and also when an individual is officially representing the project in public spaces.

## Enforcement

Instances of behavior that violate this Code of Conduct may be reported to the project maintainers by opening a private security advisory on this repository, or by emailing the maintainer listed in the repository's `README.md`.

All complaints will be reviewed and investigated promptly and fairly. All maintainers are obligated to respect the privacy and security of the reporter of any incident.
